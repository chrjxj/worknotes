# Setup Python Virtualenv #


## Install reference:


## Env

```
# install python 2.7 and 3
sudo apt-get install -y python-pip python3-pip python-virtualenv

# init virtualenv
virtualenv ENVNAME

# active ENV
source ./ENV/bin/activate

```
